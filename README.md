# Tralalela — Трекер учебных задач и командных проектов(Учебный проект по производственной практике)

**Tralalela** — это веб-приложение для студентов, которое объединяет личный трекер дедлайнов, канбан-доски для групповой работы и общую панель мониторинга успеваемости в одном простом интерфейсе.

Приложение построено по модульной структуре и позволяет координировать как индивидуальную учёбу (лабораторные, домашние работы), так и совместные студенческие проекты.

---

## 🚀 Основной функционал

* 🔐 **Вход и совместная работа:** Регистрация/авторизация пользователей, создание учебных групп и добавление одногруппников по ссылкам-приглашениям.
* 📊 **Главная панель (Дашборд):** Сбор оперативной информации на одном экране — ближайшие дедлайны, активные проекты и наглядные графики успеваемости.
* 📋 **Канбан-доски:** Полноценный инструмент для командных проектов. Поддерживает разделение на этапы, управление карточками задач и их перемещение между колонками (Drag-n-Drop).
* 📚 **Дисциплины и дедлайны:** Интерактивные списки предметов и задач с возможностью фильтрации по типу работы (одиночное/командное) и мгновенным обновлением статуса выполнения через чекбоксы.
